🎯 SwingFund – Golf, Rewards & Charity Platform
SwingFund is a modern full-stack web application that combines golf performance tracking, monthly reward draws, and charity contributions into one engaging platform. It is designed with a clean, emotion-driven UI that focuses on impact rather than traditional golf aesthetics.

URL :-https://ais-pre-xdvaxf34lt2zy5imfomvj2-669348230654.asia-southeast1.run.app

🚀 Features
👤 User Features


📝 Enter and manage last 5 golf scores (Stableford format)


🎯 Participate in monthly reward draws


❤️ Choose and support a charity


📊 View dashboard with scores, participation & winnings


🔐 Secure authentication system


🏆 Reward System


🎲 Monthly draw system


🥇 5-number match (jackpot with rollover)


🥈 4-number match


🥉 3-number match


⚙️ Random / algorithm-based draw logic


💰 Subscription System


Monthly & yearly plans


Secure payment integration


Access control for subscribers


🛠️ Admin Features


👥 Manage users & subscriptions


🎯 Configure and run draws


❤️ Manage charity listings


✅ Verify winners and payouts


📈 Analytics & reports

2️⃣ Install dependencies

3️⃣ Setup environment variables
Create a .env file:
PORT=5000MONGODB_URI=your_mongodb_uriJWT_SECRET=your_secret_key
